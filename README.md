# WORPL 쪽지 오토클릭

WORPL 받은쪽지함의 신규 쪽지를 전용 Chrome 창에서 자동으로 클릭해 읽음 처리하고, 결과를 엑셀 파일로 저장하는 Windows 데스크톱 프로그램입니다.

## 제작자

- 이름: 박재민
- 이메일: jaemini@gmail.com

## 설치

사용자 설치에는 `dist\WORPL Message Reader Setup 0.1.13.exe` 파일 1개만 필요합니다.

## 개발 실행

```powershell
npm install
npm run dev
```

## 테스트

```powershell
npm test
```

## 설치 파일 빌드

```powershell
npm run build
```

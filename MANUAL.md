# WORPL 쪽지 오토클릭 설치 및 사용 매뉴얼

## 필요한 파일

- 설치에는 `WORPL Message Reader Setup x.x.x.exe` 파일 1개만 필요합니다.
- `dist` 폴더의 `.blockmap`, `latest.yml`, `win-unpacked` 폴더는 개발/빌드 산출물이며 사용자 설치에는 필요하지 않습니다.
- 구버전 설치 파일은 `dist/old-versions` 폴더에 보관합니다.

## 설치 방법

1. 최신 설치 파일을 실행합니다.
   - 예: `WORPL Message Reader Setup 0.1.4.exe`
2. Windows 보안 경고가 표시되면 게시자를 확인한 뒤 실행을 선택합니다.
3. 설치 위치를 선택하고 설치를 완료합니다.
4. 바탕화면 또는 시작 메뉴의 `WORPL Message Reader` 바로가기로 실행합니다.

## 기본 사용 순서

1. 앱을 실행합니다.
2. `Chrome 열기`를 누릅니다.
3. 열린 전용 Chrome 창에서 WORPL에 직접 로그인합니다.
4. 받은쪽지함 화면으로 이동합니다.
5. 실행 모드를 선택합니다.
   - `키워드 신규만`: 제목에 입력한 키워드가 있고, 제목 끝에 빨간색 `new` 표시가 있는 쪽지만 대상으로 합니다.
   - `신규 전체`: 빨간색 `new` 표시가 있는 신규 쪽지를 지정한 수량까지 대상으로 합니다.
6. `미리보기`를 눌러 대상 목록을 확인합니다.
7. `읽음 처리 실행`을 누릅니다.
8. 완료되면 `쪽지 읽기 XX건 완료` 메시지가 표시되고 엑셀 파일이 저장됩니다.

## 저장 파일

- 기본 저장 위치는 사용자 문서 폴더입니다.
- 경로 형식:
  `C:\Users\<사용자>\Documents\WORPL Message Reader\YYMMDD_HHMM_WP쪽지_엑셀시트명.xlsx`
- `키워드 신규만`으로 실행하면 엑셀 시트 이름은 입력한 제목 키워드입니다.
- `신규 전체`로 실행하면 엑셀 시트 이름은 읽을 수량 기준의 `신규XX개`입니다.
- 저장 컬럼은 `순번`, `Date`, `작성자`, `제목`, `링크`입니다.
- 최초 대상 건수와 실제 기록 건수가 다르면 `처리 안내` 시트가 추가되고 미기록 건수와 사유가 기록됩니다.

## 주의사항

- 앱이 연 전용 Chrome 창을 사용해야 합니다.
- 기존에 쓰던 일반 Chrome 창은 자동화 대상이 아닙니다.
- WORPL 운영 사이트의 쪽지를 실제로 클릭하므로, 실행 전 미리보기 목록을 확인해야 합니다.
- 본문 내용은 엑셀에 저장하지 않습니다.
- 신규 쪽지는 제목 영역의 빨간색 `new` 글씨만 기준으로 판단합니다.

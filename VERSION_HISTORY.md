# WORPL 쪽지 오토클릭 버전 히스토리

## 0.1.19 - 2026-06-25

- 읽음 처리 중 WORPL 검색 결과가 갱신되어 `data_id`가 있는 대상 쪽지가 현재 검색 결과에서 사라지면 전체 실행을 중단하지 않고 안전하게 스킵하도록 변경했습니다.
- 같은 실행 배치 안에 이미 처리한 `data_id`가 다시 나타나면 중복 클릭하지 않고 건너뜁니다.
- 다른 쪽지를 대신 클릭하지 않도록 `data_id`가 확인된 대상만 스킵 처리합니다.
- 오류 로그에 실패 대상과 스킵 대상 목록을 추가로 기록하도록 보강했습니다.
- 실행 오류 후 화면에 남은 이전 미리보기 목록을 초기화해 후속 `openPreviewMessage` 오류가 발생하지 않도록 변경했습니다.
- 0.1.18 설치 파일은 구버전 산출물로 `dist/old-versions`에 보관합니다.

## 0.1.18 - 2026-06-25

- `페이지 이동 대기 시간(초)` 기본값을 120초에서 500초로 변경했습니다.
- 이 값은 고정 지연 시간이 아니라 최대 대기 시간이므로, 정상적으로 빨리 로딩되는 경우 전체 처리 속도에는 영향을 주지 않습니다.
- 0.1.17 설치 파일은 구버전 산출물로 `dist/old-versions`에 보관합니다.

## 0.1.17 - 2026-06-24

- 기본 글자 크기를 키우고 본문, 라벨, 상태 메시지, 테이블 텍스트 색상을 검정 계열로 강화했습니다.
- 입력칸과 버튼 높이, 테이블 헤더 대비를 높여 장시간 사용 시 읽기 쉽게 조정했습니다.
- 0.1.16 설치 파일은 구버전 산출물로 `dist/old-versions`에 보관합니다.

## 0.1.16 - 2026-06-24

## 0.1.15 - 2026-06-24

- 페이지 이동 대기 시간을 사용자가 설정할 수 있도록 `페이지 이동 대기 시간(초)` 입력칸을 추가했습니다.
- 기본값은 120초이며, 30초부터 600초까지 설정할 수 있습니다.
- 사용자가 설정한 대기 시간을 받은쪽지함 진입, 검색 결과 다음 페이지 이동, 쪽지 읽기 후 목록 복귀, 미리보기/실행 흐름에 적용합니다.
- 오류 로그에 실제 적용된 페이지 이동 대기 시간 값을 함께 기록하도록 변경했습니다.
- 0.1.14 설치 파일은 구버전 산출물로 `dist/old-versions`에 보관합니다.

## 0.1.14 - 2026-06-24

- 쪽지가 2만 건 이상인 계정에서 받은쪽지함 로딩이 10초를 넘으면 `page.goto: Timeout 10000ms exceeded` 오류가 발생하던 문제를 수정했습니다.
- 받은쪽지함 진입, 검색 결과 다음 페이지 이동, 검색 결과 복귀 등 주요 내비게이션 타임아웃을 대량 계정 기준으로 120초까지 기다리도록 변경했습니다.
- 읽음 처리 중 오류가 발생하면 실제로 읽음 처리한 쪽지만 엑셀에 저장하고, 같은 저장 폴더의 `logs` 하위 폴더에 오류명, 오류 메시지, 스택, 처리 건수, 저장 경로를 남기도록 추가했습니다.
- 오류 로그는 본문, 쿠키, 계정 정보를 저장하지 않고 엑셀과 같은 범위의 쪽지 목록 메타데이터만 기록합니다.
- 0.1.13 설치 파일은 구버전 산출물로 `dist/old-versions`에 보관합니다.

## 0.1.13 - 2026-06-23

- 재검색 후 같은 쪽지의 제목 표기, 줄바꿈, 링크 URL 형태가 달라지면 `쪽지 링크를 현재 검색 결과에서 다시 확인할 수 없습니다` 오류가 발생하던 문제를 수정했습니다.
- 같은 쪽지 판별 기준을 `링크+제목` 우선에서 WORPL `data_id` 우선으로 변경해, 같은 대상만 재시도하고 다른 쪽지를 대체 클릭하지 않도록 보강했습니다.
- 재시도 확인은 현재 검색 결과 기준으로 최대 3페이지까지만 제한적으로 확인해 서버 부담을 과도하게 늘리지 않도록 했습니다.
- 0.1.12 설치 파일은 구버전 산출물로 `dist/old-versions`에 보관합니다.

## 0.1.12 - 2026-06-23

- 재시도 확인 시 신규 필터 목록뿐 아니라 현재 검색 결과의 전체 후보 행에서 같은 `링크+제목` 대상을 찾도록 변경했습니다.
- 대상이 현재 검색 결과에 실제로 보이는 경우에만 재시도하므로, 다른 쪽지로 대체 클릭하지 않습니다.

## 0.1.10 - 2026-06-23
